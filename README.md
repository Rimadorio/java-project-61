# 🧠 Brain Games

[![Java CI](https://github.com/rimador/java-project-61/actions/workflows/gradle.yml/badge.svg)](https://github.com/rimador/java-project-61/actions/workflows/gradle.yml)
[![Maintainability](https://api.codeclimate.com/v1/badges/a99a88d28ad37a79d2c3/maintainability)](https://codeclimate.com/github/rimador/java-project-61/maintainability)

**Brain Games** — это набор консольных мини-игр для тренировки ума. Проект создан в рамках обучения на Hexlet и демонстрирует основы Java, работу с Gradle, Checkstyle и Git.

## 🎮 Доступные игры

| № | Игра | Описание |
|:-:|------|----------|
| 1 | Greet | Приветствие (запрос имени) |
| 2 | Even | Определить, является ли число чётным |
| 3 | Calc | Вычислить результат арифметической операции |
| 4 | GCD | Найти наибольший общий делитель двух чисел |
| 5 | Progression | Найти пропущенное число в арифметической прогрессии |
| 6 | Prime | Определить, является ли число простым |

## 🚀 Быстрый старт

### Требования

- Java 21 или выше
- Gradle 9.0 (или использование Gradle Wrapper)

### Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/rimador/java-project-61.git
cd java-project-61

# Перейти в директорию приложения
cd app

# Собрать проект
make build

# Запустить игры
make run
